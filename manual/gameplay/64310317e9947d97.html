    0. 模组安装教程 Mod Installation Tutorial

    Note: When downloading files, be mindful of your operating platform and distinguish between mod packages for iOS, Android, and Windows platforms!

    注意：下载文件时候，注意自己的运行平台，注意区分 iOS，Android，Win 平台的模组包！

    Installation 安装方式

    Android 安卓平台

    After downloading the mod file, select our game as the application to open it.

    模组文件下载后，打开方式选择我们游戏。

    批量安装模组方式：

    注意，一次不要选太多模组，选十多个就进一次游戏，以确保模组正常安装。

    Windows 电脑平台

    Move the mod file to one of the following locations:

    • Standard Edition: steamapps\common\PanzerWar-DE\dlc\PanzerWar\mods\Installs
    • Definitive Edition: steamapps\common\PanzerWar-DE\dlc\PanzerWar-DE\mods\Installs
    • HD Test Version: steamapps\common\PanzerWar-DE\dlc\PanzerWar-DE-HDRP\mods\Installs

    Then, start the game.

    将模组文件移动至

    普通版：steamapps\common\PanzerWar-DE\dlc\PanzerWar\mods\Installs

    或决定版：steamapps\common\PanzerWar-DE\dlc\PanzerWar-DE\mods\Installs

    或高清测试版：steamapps\common\PanzerWar-DE\dlc\PanzerWar-DE-HDRP\mods\Installs

    再启动游戏即可

    iOS 苹果平台

    In your system's file manager, manually move the downloaded modpack to the mods/Installs directory within our game's folder.

    For the Definitive Edition, the folder is named "Panzer War," and for the Standard Edition, it's named "WarOfTanks Arcade."

    在系统的文件应用中，将下载的 modpack 手动移至我们的游戏目录下的 mods/Installs 目录中

    决定版叫 Panzer War ，普通版叫 WarOfTanks Arcade
