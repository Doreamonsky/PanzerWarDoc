    0. Tank Workshop Basic Tutorial 坦克工坊通用教学

    ShortKey 快捷键

    W，A，S，D to move the camera forward, backward, left, and right 控制摄像机前后左后移动

    Q，E to move the camera up and down 控制摄像机上下移动

    1 - Slot Mode 插槽模式

    2 - Move 移动

    3 - 旋转

    4 - Rotate 缩放

    Ctrl + Z - Undo 上一步

    Ctrl + Y - Redo 下一步

    Ctrl + C - Copy 复制

    Ctrl + V - Paste 粘贴

    Ctrl + X - Cut 剪贴

    Change Vehicle Hull 更换车身

    Select the turret you want to use, then click Copy

    选中你要使用的炮塔，然后点击拷贝

    Click Paste

    点击粘贴

    Test Drive 试驾

    First, save the vehicle, then click Test Drive

    首先保存载具，然后点击试驾

    Click Stop Act to return to the Tank Workshop

    点击停止扮演，返回坦克工坊
