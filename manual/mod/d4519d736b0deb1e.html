    35. Custom ui interfaces

    Before start the tutorial. You should learn the lua development of Panzer war first

    22. Lua 开发

    Create ui prefab

    Create a ui prefab

    To generate lua view, you need to set prefix for objects

    #btn_ for Button component

    #img_ for Image component

    #input_ for InputField component

    #txt_ for Text component

    #obj_ for GameObject

    Generate lua view

    The ui code consist of two parts. The view code and controller code.

    Most of the view code can be generated by our tool

    And controller code can only be created by yourself.

    You can look and refer our codes at folder LuaProject/user/lib/ui

    Then lua view is pasted into your clipboard. You can press ctrl+v in the vs code

    Create ui resource config

    Pick your ui resources from here

    Click on the Pack all configs. The byte files will be generated at Build\Config

    Paste the byte file into the mod directory

    Load ui in game

    You can generate the ui through the following code. replace content in <> to your contents.

    LuaUIManager.CreateUI(<YourViewInstance>, <YourControllerInstance>,
            <ConfigGuid>,
            <PrefabName>)
