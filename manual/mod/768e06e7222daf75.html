    02.16 Hatch Compatible - 载具舱盖适配

    创建空物体，挂 VehicleHatchAttachmentMonoComponent

    Create an empty GameObject and attach the VehicleHatchAttachmentMonoComponent.

    Double Side Rendering 双面渲染

    舱盖打开时候，需要显示背面的部分需要设置为双面。 注意，此选项会翻倍三角面，请近可能分离模型，使只有活动区域生成背面模型。

    When the hatch is open, the back side needs to be set as double-sided. Note: enabling this option will double the triangle count. Please separate the model as much as possible so that only the moving parts generate the back side geometry.

    在设置完成双面模型后，点击烘培按钮，以生成模型文件。

    After setting the double side model , click bake button to create model file.
