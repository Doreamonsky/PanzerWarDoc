    0.3.4 New Water Integrate 新水体集成

    启用波浪效果需导入此包

    To enable the wave effect in the selected content, import this package.

    Pick 选择** PanzerWarMods/com.windyverse.urp.water/package.json**

    If the folder com.windyverse.urp.water not exists, then download the latest tool

    若不存在 com.windyverse.urp.water 文件夹，则需下载最新工具

    启用实体波浪需在渲染管线中开启此选项，并将游戏内水体调整到极致
