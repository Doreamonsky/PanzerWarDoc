    Custom Module Presets 自定义模块预设

    Create module folder 创建你的模型文件夹

    Create a folder and create a sub folder （Name as internal module type） Then import your fbx model.

    创建一个文件夹，并创建个子文件夹（命名为模块类型名称）。然后导入你的模型的 fbx。

    Sub folder is names with internal module. The following is some options and you can refer preset files at Packages/com.shanghaiwindy.middlesharedresources/SharedInternalDamage/Preset folder.

    子文件的名称以模块类型命名，有如下几种选项。也可参考位于 Packages/com.shanghaiwindy.middlesharedresources/SharedInternalDamage/Preset 的项目内预设文件

    Fill-in the folder you created. It is started with Assets/

    填入你的创建文件夹的路径，一般都是以 Assets/ 开头的

    Use your internal modules 使用你的内构模型

    Under the Vehicle HitBox Editor, click on refresh button and select the correct model type. You can use your own internal modules.

    在伤害模型编辑器中，点击刷新预设，并选择正确的模块类型，你就可以看到自己导入的内构模型。
