    26.2. Setup Fmod Proj 设置 Fmod 工程

    Make sure you have finished this step.

    确保你已经完成了此步骤。

    1. GetStarted 项目初始化

    1. Import to FMod Studio 导入 FMod Studio

    **Plan A: Open the pre-configured FMOD project directly and build the banks.
    **You can then proceed to Step 2.

    方案.A 直接打开已配置完成的 Fmod 工程，然后打包 Bank，

    即可进入步骤 2

    Project download link:

    工程下载地址： https://www.123865.com/s/8pZNTd-6dSKA

    Plan B: Import using an empty project:

    方案.B 导入空项目的方式：

    Create a empty folder name it as what ever you want. The name is FModUGCFolder in our demonstrate.

    创建一个任意名字的空文件夹，在本次演示中，我使用了 FModUGCFolder 的名称作为文件夹名称

    2. Unity Editor Settting Unity 编辑器设置

    When you see all the events, it means everything is imported correctly.

    当你看到你的事件名称，这就表明工程已正常设置
