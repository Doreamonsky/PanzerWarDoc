    01.6 Mod Env Setup 模组环境搭建 (Hub Env)

    Install Unity Hub 安装 Unity Hub

    Install Unity Engine 安装 Unity 引擎

    If Unity Hub is installed correctly, the engine installation screen will appear. In the modules section, select only Android Build Support and iOS Build Support. Other modules are not required!

    如果你正确的安装了 Unity Hub ，则会弹出引擎安装界面，在模块中仅勾选 Android Build Support 与 iOS Build Support 。其他模块无需安装！

    Create Unity Project 创建 Unity 工程

    Click Create Project

    点击 创建工程

    Pick a empty folder

    选择一个空文件夹

    Copy the generated project path

    拷贝创建的工程路径

    Open the UnityHub and add the project folder by clicking the add button

    打开 UnityHub 并添加工程文件夹，点击 Add 按钮

    Pick add project from disk

    选择从本地添加工程

    Click to open the project

    点击打开工程

    After opening the project, please complete the following settings.

    在工程打开后，请完成以下设置

    1.4 设置 Steam 游戏资源路径 - Set Steam Game Folder

    Start Your Creation 开始你的创作

    Choose a suitable tutorial to get started based on your creative needs.

    根据你的创作需求，选择合适的教程作为开始。
