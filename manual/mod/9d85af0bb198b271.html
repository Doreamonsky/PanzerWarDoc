    22.5.6 载具时序

    如果是直接挂在载具上的Lua 脚本，请不要在 OnStarted 上直接运行初始化代码。

    请先注册一个加载回调，如下方式：

    local ret, vehicle = self.script:TryGetTankInitSystem()
if not ret then
  return
end

VehicleAPI.RegisterVehicleLoadedEvent(vehicle, function()
    -- 这里填写原本的初始化逻辑
  end)
