    02.12 HitBox Naming 伤害模型命名规范

    Naming 命名规范

    There is a naming rule that can automatically set hitbox armor in Unity.

    Normal Armor: Name_{ArmorValue}

    Intermittent Armor: Name_{ArmorValue}_Inter

    Replace {ArmorValue} to armor number such as 100

    对于伤害模型，有一个命名规范，会在 Unity 内自动设置装甲数值

    普通装甲： 名称_{装甲值}

    间隙装甲： 名称_{装甲值}_Inter

    替换 {装甲值} 为你想要的数值 比如 100

    Internal Module 内构模型 命名规范

    There is a naming rule that can automatically set internal module in unity

    {ModuleType}**{HitPoint}{ArmorValue}**

    Example:

    Transmission_120_15

    If duplicate items in blender you can name with .001 .002

    eg.

    AmmoShell_50_15.001

    AmmoShell_50_15.002

    对于内构模型，有一个命名规范，会在 Unity 内自动设置模块类型

    {模块类型}**{血量}{装甲值}**

    例子：

    Transmission_120_15

    如果在 Blender 建模软件内存在重名的情况，你可以以 .001 .002 结尾

    如：

    AmmoShell_50_15.001

    AmmoShell_50_15.002

    {ModuleType} {模块类型} 备注 (仅供参考，建模软件中请填写左侧的英文名称)
    Transmission 传动
    Engine 引擎
    HeatSink 散热器
    OilTank 油箱
    Traversing 方向机
    Breech 炮闩
    Elevator 高低机
    Sighting 观察镜
    AmmoShell 弹药架
    Crew 成员
    InternalArmor 内部装甲
    Radiator 雷达
    AutoReloader 自动装弹机
    OilPipline 滑油系统
    SurfaceControlRope 舵面控制钢索
    WingSurface 机翼

    If the crew module type is Crew. It should end with its crew type or crew types. _{CrewType}

    Examples:

    Crew_10_0_Commander_Gunner

    Crew_10_0_Radioer

    duplicate items in blender you can name with .001 .002

    eg.

    Crew_10_0_Reloader.001

    Crew_10_0_Reloader.002

    如果模块类型为Crew (成员)，则可根据其职业，

    以** _{职业类型}** 结尾

    例子：

    Crew_10_0_Commander_Gunner

    Crew_10_0_Radioer

    如果在 Blender 建模软件内存在重名的情况，你可以以 .001 .002 结尾

    Crew_10_0_Reloader.001

    Crew_10_0_Reloader.002

    {CrewType} 备注 (仅供参考，建模软件中请填写左侧的英文名称)
    Commander 指挥官
    Driver 驾驶员
    Gunner 炮手
    Radioer 通讯员
    Reloader 装填手
