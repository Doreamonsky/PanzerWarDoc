    02.13 Blender Axis Fix 轴向修正

    输入 R，X，-90，然后按回车。

    Input R, X, -90, and press Enter.

    按 Ctrl + A，选择 应用旋转。

    Press Ctrl + A to select Apply Rotation.

    再次输入 R，X，90，然后按回车。

    Input R, X, 90, and press Enter again.

    接着导出 FBX 并导入 Unity，载具的轴向将与 Unity 的轴向保持一致。

    Then export the FBX and import it into Unity. The vehicle's axis will align with Unity's axis.
