    31. Instalod Plugin 自动化 LOD 接入

    LOD

    用于降低远距离的面数，提升游戏渲染性能。

    Install the package to editor 将 Package 装到 Editor 上

    Click on + and select Add package from disk ... to add this local package from Panzer War Mod folder 点击 + 选择 Add package from disk ... 从装甲纷争的模组文件夹中选择这个 json 文件

    Register account 注册账号

    在此处注册账号，并购买免费资格 https://instalod.com/licensing/

    获取资格后，打开 Unity 此窗口，登录你的账号

    Toggle on Instalod support in editor 启用 Instalod 支持

    Then repack you vehicles. Your vehicles will use lods generated from Simplygon.

    然后点击载具上的 Repack，你的载具会使用由 Instalod 提供的 Lod 模型

    View the lod 查看生成的 lod

    Lod 默认生成在 Assets/Res/Cooked/SimplifiedMeshGroup 目录下
