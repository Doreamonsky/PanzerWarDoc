    10. Encryption 加密模组

    The encryption mod is a way to encourage modders to create high quality mods. You can only create the encryption mods to player if you created your mod without copyright issues. In other word,you can't use models from other games such as World of Tanks / War Thunder to make encryption mods. If you make encryption mods with copyright issues,the developer has rights to close your account and revoke all the keys you have generated. This agreement will affect all the mod package that is created after 2020/7/5. You can purchase models from CGTrader / Unity asset store and other cg stores to create encryption mods. Or you can learn to create model on your own.

    加密模组是一种鼓励模组制作者创造高质量模组的方式。只有在确保自己所制作的模组没有版权问题的情况下，才能为玩家创建加密模组。换句话说，你不能使用其他游戏（如《坦克世界》/《战争雷霆》）的模型来制作加密模组。如果你使用有版权问题的内容制作加密模组，开发者有权关闭你的账户并撤销你生成的所有密钥。该协议将影响2020年7月5日之后创建的所有模组包。你可以从CGTrader、Unity资产商店及其他CG商店购买模型来制作加密模组，或者学习自己创建模型。

    Bundled Sub ModPack 捆绑包


    The module now supports the addition of bundle functionality. For example, if Module A is set to bundle the UIDs of Modules B and C, when a user activates Module A, they will automatically receive Modules B and C as well. Additionally, you can dynamically adjust the bundle for Module A, and the changes will be synchronized in real-time for the users.

    模组现支持增加捆绑包功能。
    比如 A 模组设置捆绑 B，C 的 Uid 后，
    用户激活 A 模组后，自动获取 B，C 模组。
    且你可以动态调整 A 的捆绑包，用户侧会实时同步结果。

    Payment Link 支付链接


    The module now supports setting a payment link. After selecting the payment link, it will be directly displayed on the module's guided unlock page.

    模组现已支持设置支付链接，选择支付链接后，会在模组引导解锁页面，直接显示支付链接
