    01.1 Unity Engine Requirement Unity 引擎版本要求

    The following softwares are required:

    1. (Must be the same version)

    You can install Unity from this link:

    https://unity.com/releases/editor/whats-new/2022.3.62f1#installs

    To install unityhub, use this link https://public-cdn.cloud.unity3d.com/hub/prod/UnityHubSetup.exe

    If you have unity hub installed, you can install by this link:

    unityhub://2022.3.62f1/4af31df58517

    1. （需要相同版本）

    你可以从这里安装 Unity:

    https://unity.com/releases/editor/whats-new/2022.3.62f1#installs

    安装 unity hub，请使用此连接 https://unity.cn/releases

    如果你已经安装了 Unity Hub 则可以在游览器中输入以下连接安装

    unityhub://2022.3.62f1/4af31df58517

    或者 win + r ，输入以上内容，点击确定

    自 2025年8月下旬起，若你在中国大陆地区，则因 Unity 中国的限制，无法正常下载国际版。可通过我们提供的网盘下载 Unity 国际版引擎 网盘：https://www.123865.com/s/8pZNTd-dR8KA

    You will only need to install Android Build Support & iOS Build Support. Do not install VS 2022 and OpenJDK,Android SDK.

    你只需要安装 Android Build Support 与 iOS Build Support
